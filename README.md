# Sistemas Distribuídos

Projeto de sistemas distribuídos desenvolvido em Java.

## Descrição

Este projeto implementa um sistema cliente-servidor para gerenciamento de contas bancárias, desenvolvido como parte dos exercícios práticos (EP-2 e EP-3) da disciplina de Sistemas Distribuídos.

## Tecnologias Utilizadas

- Java
- Maven
- SQLite (database.db)

## Estrutura do Projeto

```
sistemas_distribuidos/
├── src/
├── target/
├── database.db
├── pom.xml
└── README.md
```

## Como Executar

1. Clone o repositório
2. Compile o projeto usando Maven
3. Execute o servidor
4. Execute o cliente
